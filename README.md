# a vue component ->  cus-avatar


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at
localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
源码文件 /src/index
打包后文件 /dist/index
